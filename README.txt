Steps to Play:
Open index.html, make sure the CCS, Images and JS folders are present as well.

Option:
If you want different character Refresh page and a random character will show up on screen

How to play:
Try and move character using the arrow keys on keyboard. Move Left, Right, Up, and Down to avoid the bug as you try and get to the water at the top of the screen.

To Finish game, close web browser.
